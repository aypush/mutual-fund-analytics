# Mutual Fund Analytics Project

## Overview

This project analyzes Indian mutual fund industry data to generate insights on fund performance, investor behavior, SIP growth, AUM trends, portfolio allocation, and risk-adjusted returns.

## Objectives

* Perform exploratory data analysis (EDA) on mutual fund datasets.
* Analyze SIP inflows, AUM growth, investor demographics, and category trends.
* Compute performance metrics such as CAGR, Sharpe Ratio, Sortino Ratio, Alpha, Beta, and Maximum Drawdown.
* Build a composite Fund Scorecard for ranking mutual funds.
* Compare top-performing funds against benchmark indices.

## Project Structure

### Source Code

Python scripts for data ingestion, cleaning, EDA, database creation, and analytics.

### Datasets

Raw and processed mutual fund datasets used throughout the project.

### Documentation

Project reports, insights, charts, and findings.

### Notebook

EDA_Analysis.ipynb contains the complete exploratory data analysis workflow.

## Key Insights

* SIP inflows showed strong growth from 2022–2025.
* Equity-oriented categories attracted the highest inflows.
* Working-age investors (26–45) formed the largest investor segment.
* Top fund houses dominated industry AUM.
* Risk-adjusted performance metrics helped identify top-performing funds.
