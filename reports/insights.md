# Mutual Fund Analytics Insights

## 1. Category Performance
Some categories consistently give higher returns than others, showing category selection is important.

## 2. Top Funds
Only a small number of funds contribute most of the high returns.

## 3. Fund House Analysis
Certain fund houses show consistently better performance across schemes.

## 4. Expense Ratio Effect
Expense ratio does not strongly determine returns in this dataset.

## 5. Key Conclusion
Returns are more dependent on category selection than individual fund selection.
